Featured Project

💊 MEDIBUDDY — Smart Medicine Reminder & Expiry Tracker

A modern web application designed to help patients and families manage medications, improve adherence, and reduce health risks caused by missed doses or expired medicines.


The Problem

  • Patients forget to take medicines on time.
  • Expired medicines remain in households.
  • Medication purposes become confusing.
  • Approximately 50% of chronic patients do not follow medication correctly.

Core MVP Features

  • Add medicine details.
  • Browser notification reminders.
  • Expiry tracker with colour indicators.
  • Medication adherence history.
  • Caregiver sharing mode.

Technology Stack

Frontend: HTML, CSS & Vanilla JavaScript
Storage: Local Storage
Notifications: Browser Notification API
Deployment: GitHub Pages / Netlify

Creative Features

  • 📷 AI-powered medicine box scanning.
  • 🌍 Multilingual reminders.
  • 🔔 Refill alerts before medicines run out.
  • 📈 Dashboard with adherence ring, streak counter, and medicine statistics.
