### MEDIBUDDY — Smart Medicine Reminder & Expiry Tracker

**The Real-World Problem**
- People (especially elderly patients and busy families) forget to take medicines on time.
- Households accumulate expired medicines that are dangerous to consume.
- Patients forget what each medicine is for, leading to confusion and missed doses.
- WHO estimates ~50% of patients with chronic illness don't take medication correctly.

**Core Features (MVP)**
1. Add Medicine — name, dosage, times per day, start/end date, expiry date.
2. Smart Reminders — browser/push notifications at scheduled times.
3. Expiry Alerts — flags medicines expiring within 30 days (color-coded: green/yellow/red).
4. "Taken / Skipped" tracker — adherence history with a simple streak chart.
5. Caregiver Mode — optional shareable summary for family members.

**Suggested Tech Stack**
- Frontend: HTML + CSS + Vanilla JS
- Storage: local Storage (no backend needed for MVP!)
- Notifications: Browser Notification API
- Deploy: Netlify / GitHub Pages

**Creative Extra Features**
- 📷 Scan medicine box (use image upload + AI description).
- 🌐 Multilingual reminders (for elderly relatives).
- 🔔 "Refill soon" alert when pills are running low.

**What's included**
- Core features: Add medicines, localStorage persistence, Today's Schedule with Mark as Taken toggle, browser notifications, Expiry Tracker (Red/Yellow/Green)
- Dashboard: Animated adherence ring, Day streak counter, Medicines tracked count
